🌐 Custom Domain

📋 Como Usar o Domínio Customizado

✅ O domínio funciona DIRETO no servidor MIRIDIAN!
Não precisas exportar nem fazer deploy separado. Basta configurar o DNS.

Passo 1: Guarda o Domínio

• Insere o teu domínio no campo acima (ex: www.mysite.com)
• Clica em "Save Domain"

Passo 2: Identifica Onde o MIRIDIAN Está Hospedado

🚂 Se estás no Railway / Render / Heroku:
• Não precisas de IP!
• Usa o domínio automático (ex: miridian.up.railway.app)
• Ou configura direto no dashboard do provider

☁️ Se estás num VPS (DigitalOcean, AWS, Linode):
• Descobre o IP: curl ifconfig.me
• Ou verifica no painel do provider
• Exemplo: 203.0.113.45

💻 Se estás em localhost:
• ⚠️ Domínios públicos NÃO funcionam com localhost
• Faz deploy primeiro (Railway, Render, VPS, etc)

Passo 3: Configura DNS

No teu registrador de domínio (GoDaddy, Namecheap, Hostinger, etc):

Se estás no Railway/Render/Heroku (usa CNAME):
Tipo: CNAME
Nome: www
Valor: miridian.up.railway.app (ou o domínio do teu provider)
TTL: 3600

Se estás num VPS com IP (usa A):
Tipo: A
Nome: @ (ou www)
Valor: 203.0.113.45 (o IP do teu servidor)
TTL: 3600

💡 Dica Railway: Podes configurar direto no Railway Dashboard → Settings → Domains

Passo 4: Aguarda Propagação

• Normalmente leva 15-30 minutos
• Pode demorar até 48 horas em alguns casos
• Testa acessando www.mysite.com

Como Funciona:

• www.mysite.com → Mostra o teu site
• www.mysite.com/page/about → Páginas internas
• /s/miridian → Continua funcionando normalmente

💡 Dica: Para exportar e hospedar independente, usa o botão "📦 Export Standalone".
